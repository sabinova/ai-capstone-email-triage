# Component 1 – Email Ingestion, Classification & Routing  

**Owner: Rimsha Tahir**

## Description
This component is responsible for receiving incoming emails, preprocessing and cleaning the text data, classifying emails into predefined categories, and routing them to the appropriate department or workflow.  

The goal of this component is to automate the initial triage process and improve response efficiency by reducing manual sorting.

## Tools & Technologies  
- n8n
- Hugging Face API
